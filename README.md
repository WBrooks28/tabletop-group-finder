# Tabletop Group Finder

## Summary 
This is an online game hosting website for tabletop games. You can post games and find games. You have to have an account with our website to be part of the games. 

## Built With 
* React.js
* GraphQL
* Node.js
* MongoDB
* JWT authrntication 
* Heroku 
* Bootstrap 
* HTML
* Javascript
* Bootstrap
